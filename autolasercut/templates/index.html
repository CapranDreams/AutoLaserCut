Laser Cutter Toolpath Helper

This application will help you convert images into laser cutting toolpaths. Additionally, there are some basic parametric templates for box making and gear creation to speed up these common applications.

This tool is developed as part of my SEIS 739 class project

Tracing Images

To trace an image, go to the 'laser cutting' tab. Tracing will give you the outer perimeter of distinct colors in the image. It does not work so well on photographs but can work great for converting a monocolor image to just its outline. This is particularly useful for converting shapes to stickers or laser cutouts.

This tool expects an image, a name, and some parameters for performing the trace. At this point, the parameters do not respond in real-time, but you can try uploading the image multiple times with different parameters to see the result.

The general process for doing this is receiving the uploaded image (frame 1 below) > converting that image to greyscale (frame 2) > thresholding that image (frame 3, multiple possibilities) > converting that image to the expected color palette for the laser cutter (frame 4)

Engraving Images

To engrave an image, go to the 'laser engraving' tab. Engraving will laser burn an image based on the greyscale intensity of the image. It works well on photographs but can only show the greyscale version of that image when burning the image onto your surface.

This tool expects an image, a name, and some parameters for performing the rasterization. At this point, the parameters do not respond in real-time, but you can try uploading the image multiple times with different parameters to see the result. It is important that you use reasonable values for the resolution. This is the scale of dithering. A value of 400 gives fast and resonable results. A value of 1200 will take a few seconds but very good resolution. A maximum limit of 4000 is imposed due to computation time.

The general process for doing this is receiving the uploaded image (frame 1 below) > converting that image to greyscale (frame 2) > dithering that image (frame 3) OR recoloring that image to the expected color palette for the laser cutter (frame 4)

Box Generation

This application will help you generate a laser cut box from basic template. Adjust the width, length, and depth of the box to generate your custom box.

Gear Generation

This application will help you generate a laser cut gears from a basic template.
