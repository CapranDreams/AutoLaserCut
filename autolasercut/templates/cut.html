Laser Outline Tracer

This application will help you convert images into laser cutting toolpaths. If you don't know what parameters to use, try using 128 for threshold 1 and 255 for threshold 2. These settings will depend on the brightness of your image and which features you want to be captured. But these default settings will work great for black and white images.
